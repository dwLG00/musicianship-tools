About

This site was created with the intent of being a singular place to go to practice musicianship skills, or so-called "music soft-skills". These skills are often hard to practice on one's own, and existing tools online tend to be rigid and linear. Each game on this site is created with the intent of being a tool; they are written to be flexible enough to create custom practice formats around them. More content is scheduled to be released in the near future.
